# hello-vite-wasm-pack

Rust を WASM にコンパイルして Vite プロジェクトで呼び出すサンプルです。

## How to use

1. clone

```bash
git clone https://github.com/kanaru-ssk/hello-vite-wasm-pack.git
cd hello-vite-wasm-pack
```

2. pnpm install

```bash
pnpm install
```

2. build wasm

```bash
pnpm build-wasm
```

4. start server

```bash
pnpm dev --open
```
